$______Requisitos para echar a andar merkit:______$
---------------------------------------------------

- postgreSQL
- nodejs
- jdk
- nodemon
- yarn
- express js

--------------------------------------------------
--------------------------------------------------
$______________Para iniciar app:_________________$
--------------------------------------------------

1. iniciar server postgreSQL
2 (solo primera vez) copiar estructura de base de datos de archivo merkit_code/server/schema.sql
3. En la carpeta server colocar $ nodemon index
4. En merkit_code ejecutar $ yarn install   y   $ yarn start

**Si hay problema de permisos, colocar sudo antes de los comandos**
